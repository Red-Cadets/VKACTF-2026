## Битва экстрасенсов
| Событие | Название | Категория |
| :------ | ---- | ---- |
| VKACTF 2026 | Битва экстрасенсов | crypto |

  
### Описание


> Автор: [kustkzn]
>
Криптография это область знаний не прощающая ошибок, а поэтому криптографы
редко полагаются на магию. Однако у вас есть возможность проявить свои экстрасенсорные способности.
В данном таске используется генератор Мерсенна с генерирующимся сидом.

### Решение
Нам дан исходный код сервиса.
Открываем его и видим, что сид генерируется  следующей строчкой
```
seed = int(time.time())
```
Таким образом предугадывание сида становится простой задачей.
Пишем скрипт который использует int(time.time()) для генерации сида
Генерируем последовательность и переводим в нужный формат.

Отправляем серверу и он говорит нам флаг.

Примерный код решения предоставлен в [solver.py](solver.py)

### Флаг

```
vka{t1m3_1s_n0t_4_g00d_s33d_f0r_r4nd0mn3ss}
```
