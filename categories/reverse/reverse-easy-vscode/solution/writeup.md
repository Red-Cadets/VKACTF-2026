## rev-easy-vscode

| Событие | Название | Категория |
| :------ | ---- | ---- |
| VKACTF 2026 | Хакерские темки | reverse |

  
### Описание


> Автор: aquarius
>
Ходят слухи, что если использовать зеленую тему, то уровень скилла повышается в несколько раз.


### Решение
Дано расширение с темой для `VSCode`. После деобфускации `extension.js` видно, что методом `LSB` из лого расширения извлекается и запускается js код. Извлекаем код сами, так же деобфусцируем. В деобфусцированном пейлоде кроме кода сбора информации есть blob, который нужно расшифровать и получить флаг.

Солв представлен в файле [solve.py](solve.py)

### Флаг
```
vka{beware_of_malicious_vscode_extensions}
```
